# 01 — Literature Review

Systematic review and gap analysis. Positions the work against the maturity-model
lineage (CMMI, DAMA-DMM) and adjacent theory (complex adaptive systems, cybernetics
/ viable system model, digital twins, organizational learning, meta-/transfer learning).

**Crux deliverable:** a written answer to *"isn't this just CMMI relabeled?"* —
the gap that justifies the whole thesis.
